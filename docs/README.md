# NPF Development Platform

## Обзор проекта

**NPF Development Platform** — веб-приложение для планирования и отслеживания развития корпоративного блока Негосударственного пенсионного фонда (НПФ). Система управляет двумя параллельными треками развития:

1. **Трек 1: Сотрудники Банка** — масштабирование программы для сотрудников (текущее проникновение 9%, цель — 17%)
2. **Трек 2: Внешние клиенты** — построение корпоративного канала с нуля (целевой пул 30-50 предприятий, план сборов 3 млрд руб.)

## Быстрый старт

### Требования

- Docker Desktop
- Node.js 20+ (для локальной разработки)
- Python 3.11+ (для локальной разработки)
- Ollama (для локального LLM)

### Запуск через Docker

```bash
# Разработка (с hot-reload)
docker compose -f docker-compose.dev.yml up --build

# Production
docker compose up --build
```

### Локальный запуск

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# Frontend (в отдельном терминале)
cd frontend
npm install
npm run dev
```

### Доступ к приложению

| Среда | Frontend | Backend API | Ollama |
|-------|----------|-------------|--------|
| Development (Docker) | http://localhost:3100 | http://localhost:8100 | http://localhost:11435 |
| Development (Local) | http://localhost:5173 | http://localhost:8000 | http://localhost:11434 |

## Основные возможности

- **Дашборд** — KPI, воронка продаж, матрица рисков, диаграмма проектов
- **Управление предприятиями** — CRM для корпоративных клиентов
- **Дорожная карта** — визуализация и управление задачами по кварталам
- **База знаний (RAG)** — загрузка документов, векторный поиск, AI-ассистент
- **AI-чат** — контекстные ответы на основе базы знаний

# Руководство пользователя NPF Development

## Общее описание системы

NPF Development — это веб-приложение для планирования и отслеживания развития корпоративного блока Негосударственного пенсионного фонда (НПФ). Система предназначена для управления продажами корпоративных пенсионных программ и отслеживания ключевых показателей эффективности.

### Основные продукты НПФ

- **ПДС (Программа долгосрочных сбережений)** — государственная программа накоплений с софинансированием
- **КПП (Корпоративная пенсионная программа)** — программа для работодателей по обеспечению сотрудников дополнительной пенсией

## Разделы системы

### 1. Дашборд (Главная страница)

Дашборд отображает ключевые показатели эффективности (KPI) по трём направлениям продаж:

#### Направление 1: Внешние продажи
- **Взносы** (ключевой KPI) — сумма привлечённых взносов в млн рублей
- **Предприятия** — количество предприятий-клиентов
- **Договоры** — количество заключённых договоров
- **Участники** — количество участников программ

#### Направление 2: КПП в Банке
- **Участники** (ключевой KPI) — количество участников программы среди сотрудников Банка
- **Проникновение** — процент охвата сотрудников Банка
- **Взносы работников** — взносы сотрудников Банка
- **Взносы Банка** — софинансирование от Банка-работодателя

#### Направление 3: Продажи в ЗК (зарплатным клиентам)
- **Количество ДДС** (ключевой KPI) — количество договоров долгосрочных сбережений
- **Сумма взносов** — объём привлечённых взносов

#### Элементы дашборда
- **KPI-карточки** — показывают текущее значение, цель, процент выполнения и прогресс-бар
- **Timeline задач** — диаграмма Ганта с задачами по кварталам
- **Pipeline предприятий** — воронка по категориям A/B/V/G
- **Воронка продаж** — этапы от первого контакта до запуска
- **Матрица рисков** — визуализация рисков проекта
- **Ключевые вехи** — важные даты и события

#### Настройки дашборда (только для администратора)
- Целевые значения KPI
- Численность Банка (для расчёта проникновения)
- Правила скоринга предприятий
- Настройки матрицы рисков
- Ввод оперативных данных продаж

### 2. Предприятия (CRM)

Раздел для управления базой корпоративных клиентов.

#### Карточка предприятия
- **Название** — наименование организации
- **ИНН** — идентификационный номер налогоплательщика
- **Холдинг** — группа компаний
- **Отрасль** — сфера деятельности
- **Численность** — количество сотрудников
- **Проникновение ЗП** — процент сотрудников с зарплатными картами Банка
- **Категория** — приоритет (A/B/V/G)
- **Контактное лицо** — ФИО представителя
- **Email и телефон** — контактные данные
- **Менеджер** — ответственный сотрудник НПФ
- **Этап продаж** — текущий статус в воронке

#### Этапы воронки продаж
1. **В планах** — потенциальный клиент
2. **Первый контакт** — установлен контакт
3. **Переговоры** — ведутся переговоры
4. **Договор** — подписание договора
5. **Запущено** — программа запущена

#### Категории предприятий
- **A** — высший приоритет, крупные клиенты
- **B** — средний приоритет
- **V** — стандартный приоритет
- **G** — низкий приоритет

#### Функции раздела
- Создание и редактирование предприятий
- Импорт из Excel/CSV с автоматическим маппингом колонок
- Фильтрация по категории, статусу, этапу
- Сортировка и настройка колонок таблицы
- История контактов с клиентами
- Групповое удаление

---

### 3. Дорожная карта (Roadmap)

Планирование задач и активностей по развитию направления.

#### Структура задачи
- **Название** — краткое описание задачи
- **Описание** — детальная информация
- **Квартал** — период выполнения (Q1-Q4)
- **Трек** — внутренний или внешний
- **Статус** — текущее состояние
- **Зависимости** — связи с другими задачами

#### Треки
- **Внутренний** — задачи внутри организации
- **Внешний** — задачи по работе с клиентами

---

### 4. Документы

Хранилище документов с возможностью поиска.

#### Поддерживаемые форматы
- PDF — презентации, регламенты
- DOCX — текстовые документы
- XLSX/CSV — таблицы
- TXT — текстовые файлы

#### Категории документов
- Регламент
- Продукт (ПДС, КПП)
- Презентация
- Шаблон договора
- Аналитика
- FAQ
- Методика
- Инструкция

#### Ограничения
- Максимальный размер файла: 30 МБ
- Лимит хранилища: 3 ГБ

#### Функции
- Загрузка документов
- Просмотр и скачивание
- Редактирование названия и категории
- Архивирование
- Удаление

---

### 5. AI Ассистент (Чат)

Интеллектуальный помощник для ответов на вопросы.

#### Режимы работы
- **RAG включён** — ответы с учётом базы знаний (документы, регламенты)
- **RAG выключён** — общие ответы без контекста базы знаний

#### Возможности
- Ответы на вопросы о продуктах НПФ (ПДС, КПП)
- Консультации по пенсионному обеспечению
- Помощь в работе с системой
- История диалогов

#### Как задать вопрос
1. Откройте раздел "AI Ассистент"
2. Выберите режим RAG (вкл/выкл)
3. Введите вопрос в поле ввода
4. Нажмите кнопку отправки или Enter

---

### 6. Настройки LLM (только для администратора)

Конфигурация AI-моделей системы.

#### Настраиваемые модели
- **Chat** — модель для генерации ответов
- **Vision** — модель для анализа изображений
- **Embeddings** — модель для векторизации

## Роли пользователей

### Администратор (Admin)
- Полный доступ ко всем функциям
- Управление пользователями
- Настройки дашборда и KPI
- Настройки AI-моделей
- Редактирование любых записей

### Менеджер (Manager)
- Создание предприятий, задач, документов
- Редактирование и удаление **своих** записей
- Просмотр всех данных
- Использование AI-ассистента

### Просмотр (Viewer)
- Только просмотр данных
- Использование AI-ассистента
- Скачивание документов

## Часто задаваемые вопросы

### Как добавить новое предприятие?
1. Перейдите в раздел "Предприятия"
2. Нажмите кнопку "Добавить"
3. Заполните карточку предприятия
4. Нажмите "Сохранить"

### Как импортировать предприятия из Excel?
1. Перейдите в раздел "Предприятия"
2. Нажмите "Импорт"
3. Выберите файл Excel или CSV
4. Проверьте маппинг колонок
5. Подтвердите импорт

### Как загрузить документ?
1. Перейдите в раздел "Документы"
2. Нажмите "Загрузить документ"
3. Выберите категорию
4. Выберите файл (до 30 МБ)
5. Документ появится в списке

### Как изменить целевые KPI?
1. На дашборде нажмите "Настройки" (только для администратора)
2. Перейдите на вкладку "KPI и цели"
3. Введите новые значения
4. Нажмите "Сохранить"

### Как узнать текущий этап работы с предприятием?
В таблице предприятий есть колонка "Этап", показывающая текущий статус в воронке продаж.

### Почему я не могу редактировать запись?
Менеджеры могут редактировать только записи, которые они создали. Для редактирования чужих записей нужны права администратора.

### Как использовать AI-ассистента?
1. Откройте раздел "AI Ассистент"
2. Включите RAG для ответов из базы знаний или выключите для общих ответов
3. Введите ваш вопрос
4. Получите ответ от AI
